# Alpha – Car Detail Page 

A responsive car detail page built with **React (Vite + TypeScript)** and **Tailwind CSS**.  

- Image **carousel** for car photos
- **360° interactive view** of the car; supports drag/slider
- **Event price calculator**
- **Car overview** section with model, year, price, attributes
